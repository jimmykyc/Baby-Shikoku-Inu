HOW TO BUY $BSHIK

Install Trust Wallet

Open your Google Chrome browser and visit trustwallet.com. Download the Trust Wallet for your smartphone, set up your wallet and configure.

Get some ETH

As you need some ETH on any Ether Chain DEX, here you will need to buy some ETH on an exchange (CEX) like Uniswap.

Swap ETH to Ether Chain

Swap your ETH to Ether Chain. Then connect to Trust Wallet from Uniswap.

Find $BSHIK Ethereum Contract

Then, find $BSHIK token using our Eth contract address. Ask on Telegram group if not sure how to get it.

Buy $BSHIK

You can now swap Ether Chain to $BSHIK easily. Make sure to set slippage to 0.1 - 1%.
